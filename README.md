# String Case Manipulation

This Python script demonstrates two different methods of alternating case in strings.

## Features

- **Task 1**: Alternates the case of individual characters (even positions uppercase, odd positions lowercase)
- **Task 2**: Alternates the case of entire words (even words lowercase, odd words uppercase)

## Usage

Run the script and enter a string when prompted. The program will display the original string along with both transformation results.

## Example

```
Enter a string: Hello World Python
Original: Hello World Python
Task 1 Result: HeLlO WoRlD PyThOn
Task 2 Result: hello WORLD python
```

## Requirements

- Python 3.x
